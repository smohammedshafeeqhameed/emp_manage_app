# **App Name**: EmployeeZen

## Core Features:

- Employee Profiles: Create and manage employee profiles including roles, reporting managers, and contact information.
- Task Assignment: Assign tasks to employees with deadlines and priority levels.
- Project Management: Create and manage projects, associate employees, and track progress.
- Attendance Tracking: Record and manage employee attendance, including absences and vacation time.
- Daily Status Reporting: Employees can submit daily status updates on their tasks and projects.
- Working Hours: Track employee working hours.
- AI-Driven Task Prioritization: Uses an AI tool to suggest optimal task prioritization based on deadlines, dependencies, and employee availability, enhancing project efficiency.

## Style Guidelines:

- Primary color: Calm blue (#5DADE2), promoting trust and focus.
- Background color: Light gray (#F0F4F8), providing a neutral backdrop.
- Accent color: Soft orange (#F39C12), highlighting important actions.
- Body and headline font: 'PT Sans' (sans-serif) for clear and accessible text.
- Simple, professional icons for navigation and actions.
- Clean, well-organized layout for easy navigation and information access.
- Subtle transitions and animations to enhance user experience without being distracting.
